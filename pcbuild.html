Project Description

This was an assignment to make a quality PC with a limit of $550 excluding case and power supply. The PC had to be able to work for all projects in the shop and be able to run all of the necessary programs.

To complete the project we had to use PC partspicker to select appropriate parts to complete the PC. There were no specific requirements beyond that it must be able to functionand could not exceed $550. The rest was up to us. This is my Build. It mas designed to be fast and have plenty of storage, in addition I added a keyboard and mouse to replace the lesser quality ones currently in use. While in the end my build was not selected, it showed me how to use PC Partspicker. This is useful if I am ever required to design new computers for a company or just to see what is new in the world of PC parts.

Project Details

  • Case provided
  • PSU supplied
