Project Description

Cisco 2 Final project. Construct a secure network in packet tracer and prepare a a documentation package.

Project Details

  • Subnetting Scheme
  • Router Configuration
  • Switch COnfiguration
  • Security
  • Logical Diagram

Tasks Completed

  • Router and Switch Security
  • Network Verification
  • Purchase Order
  • Documentation Package

The project consisted of 3 phases, a planning phase, the construction of the network in packet tracer and the completion of a documentation package. During the planning phase the network was variably subnetted, and the routers and switches were chosen. In phase two the network was constructed in packet tracer and configuration, network and security was configured. In phase three a final documentation package was developed which includes pricing, statistics, logical and physical layouts and the running configurations. A purchase order and verification of the networks function was also included.
